# Worms

## Build process

### Requirements:

You need a recent version of node/npm installed.
Install the latest node from https://nodejs.org/en/ to get both.

#### Build steps:

After cloning project, run

1. `npm install`
1. `npm run compile`

## Running the game

After building, open worms.html in your browser.
